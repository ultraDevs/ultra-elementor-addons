== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ultra-elementor-addons` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= Is it possible to use without Elementor plugin? =

No, You must install and active Elementor plugin to make this plugin work.
